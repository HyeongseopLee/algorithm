# Algorithm

알고리즘 문제 풀이를 저장하는 곳 입니다.

## 커밋 규칙

### Keyword

- `solve`: 풀이에 성공한 새 문제 추가
- `fail`: 풀이에 실패한 새 문제 추가
- `improve`: 새로운 성공 풀이를 추가한 문제
- `docs`: readme 등 문서 수정

- 클릭 시 위키 페이지로 이동합니다.
